# Project: Product and Category Management

## Описание проекта

В этом проекте реализованы две основные сущности для работы с товарами и категориями:

1. Product — класс, описывающий продукт.
2. Category — класс, описывающий категорию продуктов.

Для класса Category автоматически подсчитывается:

- количество созданных категорий (`category_count`),  
- общее количество продуктов во всех категориях (`product_count`).  

Каждый объект категории также хранит свой список продуктов и количество продуктов в самой категории.

---

## Классы

### Product

Атрибуты:

- name (str) — название продукта
- description (str) — описание продукта
- price (float) — цена продукта
- quantity (int) — количество продукта в наличии
